# AI 助手功能说明

## 🤖 我是谁

我是 **Auto**，一个由 Cursor 开发的 AI 编程助手，专门用于帮助您进行代码开发、调试和项目管理。

---

## ✅ 我目前能做什么

### 1. 代码理解与分析

- **阅读代码库**
  - 理解项目结构和架构
  - 分析代码逻辑和依赖关系
  - 识别潜在问题和改进点

- **语义搜索**
  - 使用自然语言搜索代码
  - 找到相关功能的实现位置
  - 理解代码上下文

- **代码审查**
  - 检查代码质量和最佳实践
  - 发现潜在的 bug 和安全问题
  - 提供优化建议

### 2. 代码编写与修改

- **创建新功能**
  - 根据需求编写完整的代码
  - 实现前端、后端、数据库等各个层面
  - 确保代码符合项目规范

- **修改现有代码**
  - 修复 bug
  - 添加新功能
  - 重构和优化代码

- **多文件协调**
  - 同时修改多个相关文件
  - 保持代码一致性
  - 更新依赖关系

### 3. 项目部署与运维

- **部署自动化**
  - 创建部署脚本
  - 配置 CI/CD 流程
  - 管理服务器配置

- **服务管理**
  - 检查服务状态
  - 查看日志和错误
  - 诊断问题

- **数据库操作**
  - 创建数据库迁移
  - 设计表结构
  - 生成测试数据

### 4. 文档编写

- **技术文档**
  - 架构说明
  - API 文档
  - 部署指南

- **开发文档**
  - 功能说明
  - 使用教程
  - 问题排查

### 5. 问题诊断与修复

- **错误分析**
  - 分析错误日志
  - 定位问题根源
  - 提供解决方案

- **性能优化**
  - 识别性能瓶颈
  - 提供优化建议
  - 实施优化方案

### 6. 工具使用

- **文件操作**
  - 读取、写入、修改文件
  - 搜索和查找文件
  - 管理文件结构

- **终端命令**
  - 执行系统命令
  - 运行测试
  - 管理 Git 仓库

- **代码库搜索**
  - 使用 grep 精确搜索
  - 使用语义搜索理解代码
  - 查找特定模式
