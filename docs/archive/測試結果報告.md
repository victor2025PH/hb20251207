## 📋 建議的測試步驟

### 手動測試流程

1. **訪問網站**
   ```
   https://mini.usdt2026.cc
   ```

2. **進入 Wallet 頁面**
   - 點擊底部導航的「Wallet」

3. **點擊「發紅包」按鈕**
   - 在 Wallet 頁面找到「發紅包」按鈕
   - 點擊進入發送紅包頁面

4. **檢查遊戲規則彈窗**
   - 進入發送紅包頁面後，應該自動彈出遊戲規則彈窗
   - 檢查彈窗標題是否為「✨ 遊戲規則 ✨」
   - 檢查是否有發光特效
   - 檢查是否有「以後不再彈出」選擇框

5. **檢查按鈕**
   - 在「紅包類型」旁邊應該有「✨ 遊戲規則 ✨」按鈕
   - 懸停時應該有發光特效

## ⚠️ 注意事項

如果看不到新功能：

1. **清除瀏覽器緩存**
   - `Ctrl + Shift + Delete`
   - 選擇「緩存的圖片和文件」
   - 時間範圍選擇「全部時間」

2. **使用無痕模式**
   - `Ctrl + Shift + N` (Chrome)
   - `Ctrl + Shift + P` (Edge)

3. **清除 localStorage**
   - 打開開發者工具（F12）
   - Console 中執行：`localStorage.removeItem('dont_show_game_rules')`
   - 刷新頁面
