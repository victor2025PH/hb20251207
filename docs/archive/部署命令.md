# 部署命令

## 快速部署步驟

### 1. 確認代碼已推送到 GitHub
```bash
git log --oneline -5
```

### 2. SSH 連接到服務器並執行部署

```bash
ssh ubuntu@165.154.254.99
```

### 3. 在服務器上執行以下命令：

```bash
cd /opt/luckyred

# 拉取最新代碼
git pull origin master

# 構建前端
cd frontend
npm install
npm run build

# 重啟服務
sudo systemctl restart luckyred-api luckyred-bot luckyred-admin

# 檢查服務狀態
sudo systemctl status luckyred-api --no-pager
sudo systemctl status luckyred-bot --no-pager
sudo systemctl status luckyred-admin --no-pager
```

### 4. 或者使用快速更新腳本：

```bash
cd /opt/luckyred
bash deploy/scripts/quick-update.sh
```
