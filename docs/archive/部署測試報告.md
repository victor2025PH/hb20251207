## 如果發現問題

1. **服務未運行**：
   ```bash
   ssh ubuntu@165.154.254.99
   sudo systemctl status luckyred-api
   sudo journalctl -u luckyred-api -n 50
   ```

2. **前端構建失敗**：
   ```bash
   ssh ubuntu@165.154.254.99
   cd /opt/luckyred/frontend
   npm run build
   ```

3. **瀏覽器緩存問題**：
   - 清除瀏覽器緩存（Ctrl + Shift + Delete）
   - 或使用無痕模式（Ctrl + Shift + N）
