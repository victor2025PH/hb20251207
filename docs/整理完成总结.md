## 🎯 使用规则

### ✅ 以后创建新文件时：

1. **脚本文件** → 放在 `scripts/` 对应子文件夹
   - `.bat` → `scripts/bat/`
   - `.ps1` → `scripts/ps1/`
   - `.sh` → `scripts/sh/`
   - `.py` → `scripts/py/`

2. **文档文件** → 放在 `docs/` 或 `docs/archive/`
   - 当前使用的 → `docs/`
   - 历史/临时 → `docs/archive/`

3. **文本文件** → 放在 `scripts/txt/`
   - 除了 `requirements.txt` 和 `env*.txt`

### ❌ 不要放在根目录：

- 脚本文件（`.bat`, `.ps1`, `.sh`, `.py`）
- 文档文件（`.md`，除了 `README.md`）
- 临时文本文件（`.txt`，除了 `requirements.txt` 和 `env*.txt`）
