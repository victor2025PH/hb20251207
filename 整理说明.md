## 📝 使用说明

### 运行脚本

**Windows 批处理脚本：**
```cmd
scripts\bat\脚本名称.bat
```

**PowerShell 脚本：**
```powershell
.\scripts\ps1\脚本名称.ps1
```

**Shell 脚本（服务器）：**
```bash
bash scripts/sh/脚本名称.sh
```

### 查看文档

- 当前文档：`docs/`
- 历史文档：`docs/archive/`

## ⚠️ 重要规则

### 以后创建新文件时：

1. **脚本文件**
   - `.bat` → 放在 `scripts/bat/`
   - `.ps1` → 放在 `scripts/ps1/`
   - `.sh` → 放在 `scripts/sh/`
   - `.py` → 放在 `scripts/py/`
   - ❌ **不要放在根目录**

2. **文档文件**
   - `.md` → 放在 `docs/` 或 `docs/archive/`
   - ❌ **不要放在根目录**（除了 README.md）

3. **文本文件**
   - `.txt` → 放在 `scripts/txt/`
   - ❌ **不要放在根目录**（除了 requirements.txt 和 env 相关）
